# cs50
This repository includes my work for Harvard CS50, which I completed in Fall 2024. This class can be found for free online at this link: https://pll.harvard.edu/course/cs50-introduction-computer-science

## Week 1: C
- w1cw_hello.c (the mandatory hello world program)
- w1cw_hellofullname.c (takes in full name and greets the user)
- w1cw_agree.c (takes in yes or no inputs and processes them)
- w1cw_contacts.c (takes in contact info as user input and spits it back out)
- w1cw_compare.c (practice comparing two user inputs)
- w1cw_calculator.c (function creation practice using addition and substraction as examples) 
- w1cw_mario.c (creates nxn grid of bricks)
- w1cw_mariopyramid.c (creates a pyramid of bricks)
- w1cw_meow.c (meows as many times as the user desires)
- w1cw_meowtwo.c (a fancier version of meow.c)
