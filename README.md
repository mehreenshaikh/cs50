# cs50
This repository includes my work for Harvard CS50, which I completed in Fall 2024. This class can be found for free online at this link: https://pll.harvard.edu/course/cs50-introduction-computer-science

## Week 1: C
- hello.c (the mandatory hello world program)
- hellofullname.c (takes in full name and greets the user)
- agree.c (takes in yes or no inputs and processes them)
- contacts.c (takes in contact info as user input and spits it back out)
- compare.c (practice comparing two user inputs)
- calculator.c (function creation practice using addition and substraction as examples) 
- mario.c (creates nxn grid of bricks)
- mariopyramid.c (creates a pyramid of bricks)
- meow.c (meows as many times as the user desires)
- meowtwo.c (a fancier version of meow.c) 
- hellowhw.c (takes name as input and greets user)
- cash.c (calculates change owed in a transaction while ensuring as few coins need to be used as possible)
- mariomore.c (recreates mario pyramid)
